Elementos y ejemplos

Los elementos son sentencias que deciden la ejecución del código. Nos permiten tomar decisiones (condicionales) y repetir tareas (bucles).

1. Bucle while (Mientras)

El bucle while ejecuta un bloque de código mientras una condición específica sea verdadera. La condición se comprueba *antes* de cada repetición.

Ejemplo (JavaScript):

let contadorWhile = 0;
while (contadorWhile < 3) {
    console.log("While: El contador es: " + contadorWhile);
    contadorWhile++; // Incremento para avanzar
}

2. Bucle do...while (Hacer...Mientras)

El bucle do...while ejecuta el bloque de código al menos una vez, ya que la condición se verifica después de la primera ejecución. Luego, repite mientras la condición siga siendo verdadera.

Ejemplo (JavaScript):

let i = 5;
do {
    console.log("Do...While: Ejecutado una vez, aunque i sea " + i); 
    i++;
} while (i < 3); // La condición es falsa, pero se ejecutó primero el 'do'

3. Condicional if...else y if...else if...else

Estas estructuras permiten ejecutar código de manera condicional. Si la primera condición (`if`) es falsa, pasa a la siguiente (`else if`), y si todas son falsas, ejecuta el bloque else (el caso por defecto).

Ejemplo (JavaScript):

let puntaje = 85;

if (puntaje >= 90) {
    console.log("If/Else: Calificación: A (Excelente)");
} else if (puntaje >= 80) { // Si no es >= 90, ¿es >= 80?
    console.log("If/Else: Calificación: B (Notable)");
} else { // Si no es ni A ni B
    console.log("If/Else: Calificación: C (Aprobado)");
}

4. Bucle for (Para)

El bucle for se usa cuando se conoce de antemano el número exacto de repeticiones. Controla un contador desde un inicio, hasta una condición final, con un paso de incremento/decremento.

Ejemplo (JavaScript):

// (Inicio; Condición de fin; Paso)
for (let j = 0; j < 4; j++) {
    console.log("For: Iteración número: " + j);
}

5. Bucle forEach (Para Cada Elemento)

El concepto forEach (implementado en JS con `for...of` o métodos de Array) sirve para iterar fácilmente sobre cada elemento de una colección (como una lista o array) sin manipular los índices manualmente.

Ejemplo (JavaScript - Usando `for...of`):

const colores = ["Rojo", "Verde", "Azul"];

for (const color of colores) {
    console.log("For Each: El color es " + color);
}

6. Condicional select case (o `switch`)

La estructura switch (o `Select Case` en otros lenguajes) permite comparar una variable contra una serie de posibles valores (`case`). Es más limpio que usar muchos if...else if para la misma variable.

Ejemplo (JavaScript - Usando `switch`):

let opcion = "Abrir";
let mensaje;

switch (opcion) {
    case "Abrir":
        mensaje = "Se ha abierto el archivo.";
        break;
    case "Guardar":
        mensaje = "Se ha guardado el archivo.";
        break;
    default:
        mensaje = "Acción no reconocida.";
}

console.log("Switch: " + mensaje);
